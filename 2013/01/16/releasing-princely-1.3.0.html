Princely 1.3.0

16 Jan 2013

I’ve been doing some work on Princely the last couple of weeks and i’m happy to announce the release of version 1.3.0. My goal was to clear up any outstanding issues and get Asset Pipeline support into Princely, as well as add test support.

The following changes are included in this release: * Asset Pipeline support in Rails >= 3.1 * Decoupling the main Princely class from Rails for use in non-rails projects * Added option :disposition to make_and_send_pdf to allow PDFs to be sent as attachments or inline * Fixed invalid byte sequence error in UTF-8 with Ruby 1.9.3 (Gregory Becker) * Avoid re-registering the PDF Mimetype (causing warnings) (James Barona)

I’ve tested this release on a few applications I have access to, however if you run into any problems please lodge an issue.
